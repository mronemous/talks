What is PhoneGap?

Build your UI with HTML5

Native Web View

Phonegap wraps your HTML5 app in native code.
The native code renders your HTML5 app within a webview.

Access native functionality through PhoneGap

Phonegap acts as a bridge to native APIs via plugins.

Debugging Phonegap Apps

Desktop

Safari, FireFox, and IE have similar developer tools

HTML5 - standard browser dev tools can get used.
However many plugins do not work in the browser (cordova isn't actually running).

Ripple Emulator

                        $npm install -g ripple-emulator
                        ripple emulate --path platforms/android/assets/www
                        
Attempts to simulate cordova running in the browser.
Some plugins will work and others can be passed simulated data (geo location, accelorometer).

Creating Custom Plugins

config.xml

Javascript

cordova.exec( success, failure, "BluetoothSerial", "write", [data] )
Java
package com.megster.cordova;
public class BluetoothSerial extends CordovaPlugin {

    private static final String WRITE = "write";
    @Override
    public boolean execute(String action, CordovaArgs args, CallbackContext callbackContext) throws JSONException {
        if (action.equals(WRITE)) {
            String data = args.getString(0);
            bluetoothSerialService.write(data.getBytes());
            callbackContext.success();
        }
        ...
    }
}
